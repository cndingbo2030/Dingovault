# Features showcase

## Wikilinks

Create links to other pages with double brackets:

- Jump to the welcome guide: [[Welcome.md]]
- See advanced topics: [[Advanced.md]]
- Display text can differ from the file name: [[Welcome.md|Back to Welcome]]

## Tags

Tags start with `#` and are indexed for search. They work in prose and in list items.

This paragraph mentions #dingovault, #demo, and #local-first so you can search for them from the palette.

## TODO cycle

Use GitHub-style task bullets and cycle their state from the keyboard (see hint bar for the shortcut):

- [ ] Draft an idea #capture
- [ ] Refine the outline
- [x] Ship the Gold release

Try duplicating the pattern below and cycling TODO on the last line:

- [ ] Not started
- [ ] In progress
- [x] Done
