# Welcome to Dingovault

This is the **built-in Demo Vault**—a safe place to explore the app before you point Dingovault at your own notes folder.

- Start with [[Welcome.md]] for a quick tour of the outliner.
- See [[Features.md]] for wikilinks, tags, and task checkboxes.
- Open [[Advanced.md]] for YAML front matter and page aliases.

When you are ready, launch the app with your real vault path or set **vaultPath** in your config file (see the main README).
